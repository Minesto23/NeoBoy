# NeoBoy - Assets Directory

This directory is for ROM files and other assets.

## Structure

```
assets/
├── example.rom          # Placeholder (not a real ROM)
├── gb/                  # Game Boy ROMs
├── gbc/                 # Game Boy Color ROMs
└── gba/                 # Game Boy Advance ROMs
```

## Important Legal Notice

⚠️ **DO NOT** commit copyrighted ROM files to this repository.

This directory is `.gitignore`d to prevent accidental commits of ROM files.

## Usage

1. Place your legally obtained ROM files here
2. Load them via the emulator's ROM loader
3. Supported formats: `.gb`, `.gbc`, `.gba`

---

**Where to get ROMs legally:**
- Dump your own cartridges using a flashcart or dumper
- Download public domain/homebrew ROMs
- Purchase digital copies from legitimate sources

**NeoBoy does not condone piracy.**
