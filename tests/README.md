# NeoBoy - Test ROMs

This directory is for test ROMs to validate emulator accuracy.

## Recommended Test Suites

### Game Boy / Game Boy Color
- **Blargg's CPU Instruction Tests**
  - `cpu_instrs.gb` - Tests all CPU instructions
  - `instr_timing.gb` - Tests instruction timing
  - `mem_timing.gb` - Tests memory timing

- **Blargg's PPU Tests**
  - `dmg_acid2.gb` - DMG PPU accuracy test

### Game Boy Advance
- **mGBA Test Suite**
  - ARM7TDMI instruction tests
  - Memory tests
  - Timing tests

## Usage

1. Download test ROMs from their respective sources
2. Place them in this directory
3. Load via the ROM loader in the emulator
4. Compare output with expected results
